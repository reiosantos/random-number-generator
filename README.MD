### Setup

#### Inslall

```bash
git clone https://github.com/reiosantos/random-number-generator.git

cd random-number-generator

```

#### Prepare the server

```bash
npm install
```

#### start server

```bash
npm run start
```

Access the app on
```http request
http://localhost:3000
```
